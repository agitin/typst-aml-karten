## Basis für die Inhalte
Die Inhalte basieren auf den [Arzneimittellisten & Algorithmen des Arbeiter-Samariterbund-Bund Österreichs ](https://www.samariterbund.net/dokumentensammlung) sowie lokalen Protokollen der Gruppe Linz. Daher werden auch bewusst manche Medikamente, die in Linz nicht vorgehalten werden nicht aufgeführt.

## Technische Informationen
Dieses Projekt nutzt [Typst](https://typst.app/) zum Erstellen einer kompakten Merkhilfe für die Verabreichung von Arzneimitteln. Mittels des Typst-Compilers kann der Code in eine PDF-Datei übersetzt werden.  
Geeignete Druck-Anbieter werden in diesem Dokument ergänzt sobald sie überprüft werden konnten.
